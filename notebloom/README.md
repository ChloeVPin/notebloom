## About
Notebloom is a lightweight desktop notes app with a calm look and fast workflow. It is built with Go and Wails on the backend, and React, TypeScript, and Vite on the frontend, for quick capture, daily writing, and simple organization without clutter.

## What's Inside
- Simple, distraction-free writing space
- Automatic saving after short idle pauses
- Clean sidebar to organize multiple notes
- Flat command bar for common actions
- Frameless custom titlebar with the Notebloom logo
- Live note title in the window bar
- Export notes to `.txt`
- Word, character, and cursor position counts
- Safer writes and recovery on load
- Lightweight and fast

## Requirements
- Windows 10 or newer (64-bit)
- About 100 MB of free space

## Installation
1. Download the installer from the releases page
2. Run the `.exe` file
3. Follow the setup wizard
4. Launch Notebloom from your Start menu
